# [](#header-2) 发送到打开的

**动作编号**: 1109  
**动作名称**: 发送到打开的  
**动作作用的对象**: 选中文件  
**动作热键**: 无  
**动作鼠标手势**: 无  
**动作说明**: 复制/移动选中文件到当前打开的目录  
**动作截图**:  
  ![移动到](img1/1109.png)  
**动作内容**: run|"%B_Autohotkey%" "%A_ScriptDir%\外部脚本\文件处理\选中文件到打开的文件夹.ahk" "%CandySel%"
将选中文件的路径作为参数, 执行外部脚本文件 "选中文件到打开的文件夹.ahk", ATA 内置该动作  
**代码或详细解释**:    
选中文件, 执行动作, 显示一个当前打开文件夹的菜单, 选择菜单项将选中文件移动到相应文件夹.
